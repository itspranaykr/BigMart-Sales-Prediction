# 🛒 BigMart Sales Prediction

A Machine Learning web application that predicts product sales for BigMart stores using historical sales data. The project leverages data preprocessing, feature engineering, exploratory data analysis (EDA), and machine learning models to generate accurate sales predictions through an interactive Streamlit interface.

## 🚀 Live Demo

🔗 Streamlit App: bigmart-sales-prediction ∙ main ∙ app.py

🔗 GitHub Repository: https://github.com/itspranaykr/BigMart-Sales-Prediction.git

---

## 📌 Project Overview

BigMart Sales Prediction is an end-to-end Machine Learning project that helps estimate the sales of products based on various product and outlet attributes such as:

* Item Weight
* Item Visibility
* Item Type
* Item MRP
* Outlet Size
* Outlet Location Type
* Outlet Establishment Year
* Outlet Type

The project aims to assist businesses in forecasting product sales and making data-driven decisions.
